# IA_jogos
Jogo desenvolvido utilizando conceitos de inteligência, como agentes autônomos

## Requisitos
 - Python 3.x
 - Pygame
 
 
## Instalação

### No linux

```bash
	python3 --version
	sudo apt-get install python3-pip
	pip3 install pygame	
```

### No windows

```bash
	Baixar e instalar o python e pip
		python --version
```
